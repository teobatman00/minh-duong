Bạn cũng có thể sử dụng phương thức `println()` để in số (number).

Tuy nhiên, không giống như đoạn text, khi in số thì không được đặt số trong dấu nháy kép (Double Quotes):

<p className="tw-text-2xl">Ví dụ 1</p>

```java
System.out.println(3);
System.out.println(358);
System.out.println(50000);
```

Kết quả chương trình:

```powershell
3
358
50000
```

Bạn cũng có thể thực hiện các phép tính toán học bên trong phương thức `println()`:

<p className="tw-text-2xl">Ví dụ 2</p>

```java
System.out.println(3 + 3);
```

Kết quả chương trình:

```powershell
6
```

<p className="tw-text-2xl">Ví dụ 3</p>

```java
System.out.println(2 * 5);
```

Kết quả chương trình:

```powershell
10
```
